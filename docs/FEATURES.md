# Features

```text
TULIPWARS 2026 v0.1.7 — FEATURES
================================

SPACE / SHIP GAMEPLAY
- Anonymous persistent captain callsign and ship identity
- Nine explorable star systems
- Fuel-based warp travel
- System danger ratings and random travel events
- Ship hull, shields, fuel, credits, and cargo
- Long-range scanning for signals, salvage, traders, and aliens

TRADE / MULTIPLAYER BBS
- Station commodity exchange
- Dynamic buy and sell prices
- Persistent cargo and credits
- Anonymous station-local message boards
- Manual chat posts only
- Other active captains shown while docked at the same station
- No login API or user accounts

ALIEN COMBAT
- Turn-based hostile encounters
- Fire pulse cannons
- Divert power to shields
- Emergency retreat
- Enemy and player hull tracking
- Damage, salvage, rewards, and rescue behavior handled by the relay

AMY AUDIO
- 100% synthesized audio; no samples
- Procedural space ambience generated with AMY
- Randomized Juno-style pads that change between sessions
- AMY-generated warp, laser, damage, and interface effects

ELEMENT115 TRACKER
- Eight musical rows and sixteen steps
- Random Lost Signal pattern generation
- Adjustable notes and BPM
- Live AMY/Juno playback
- Optional live MIDI output
- Standard .MID file export
- Exported MIDI is preserved across installer updates
- Tracker remains available if the game relay is offline

TULIP WORLD SAFETY
- Read-only public lounge inside TulipWars
- No automated public messages
- No gameplay-event spam
- No automatic package announcements
- The game never calls world.post_message()

PLATFORMS
- This package is currently distributed as a Tulip Web public test
- The code is designed for Tulip Web, Tulip Desktop, and Tulip CC hardware
- Hardware release remains pending Web-build confirmation

```
