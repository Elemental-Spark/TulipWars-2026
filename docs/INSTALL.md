# Installation

```text
TULIPWARS 2026 v0.1.8 — PUBLIC INSTALL / UPDATE
================================================

WHAT THIS UPDATE FIXES
----------------------
- Physical Tulip AMY overload warning tones caused by excessive Juno voices.
- Element115 Tracker no longer leaves the background ambient synth allocated.
- Tracker playback is capped at six simultaneous Juno notes.
- Tulip Web can download exported .MID files directly through the browser.

TULIP WEB / DESKTOP
-------------------
1. Open https://tulip.computer/run/
2. Click Show code editor.
3. Upload INSTALL_OR_UPDATE_TULIPWARS_v0.1.8.py.
4. Select it and load it into the editor.
5. Click the green Run button.

The update replaces the complete game and launches it automatically.
It preserves the anonymous ship identity and all ELEMENT115_*.mid exports.

PHYSICAL TULIP ON WINDOWS
-------------------------
Open HARDWARE_UPDATE and double-click:
  UPDATE_PHYSICAL_TULIP_v0.1.8.bat

Connect the Tulip over USB first and close anything else using its serial port.

IMPORTANT
---------
- No account, password, email, OAuth, cookie, or login API is used.
- No automatic messages or files are posted to Tulip World.
- Existing custom self-hosted relay settings are preserved.

```
