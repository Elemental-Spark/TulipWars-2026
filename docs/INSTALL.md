# Installation

```text
TULIPWARS 2026 v0.1.7 — DISCORD USER TEST PACKAGE
==================================================

TulipWars 2026 is an anonymous BBS-style space game for Tulip.
This v0.1.7 package is intended for TULIP WEB TESTING before the
physical Tulip hardware release is approved.

NO TYPING OR PASTING IS REQUIRED.

QUICK INSTALL
-------------
1. Extract this ZIP.
2. Open: https://tulip.computer/run/
3. Click "Show code editor".
4. Click the blue Upload button in the Tulip files panel.
5. Choose: INSTALL_TULIPWARS_WEB_v0.1.7.py
6. Select the uploaded file.
7. Click the red load-to-editor arrow.
8. Click the green Run button.
9. Wait while the installer replaces the game and starts TulipWars.

The installer is the complete game. It does not download a TAR archive,
does not require setup commands, and does not post anything to Tulip World.

IMPORTANT TEST NOTE
-------------------
This is a public Tulip Web test build. Physical Tulip CC hardware installation
is intentionally not yet recommended until the Web build is confirmed stable.

OLD BUILDS
----------
The installer removes obsolete TulipWars package code and old installer/TAR
files from Tulip Web's local /user storage. It preserves:

- local_state.json — anonymous ship identity and local save
- ELEMENT115_*.mid — exported tracker MIDI files

Those /user files are browser-local Tulip Web storage. They are not uploaded
to Tulip World.

FIRST CONTROLS
--------------
1  Ship status and cargo
2  Navigation and space exploration
3  Station commodity exchange
4  Long-range scan and alien combat
5  Anonymous station chat
6  Element115 Tracker and MIDI export
7  Read-only Tulip World lounge
R  Refresh the current terminal
0  Return to the command deck

Ctrl+Q quits the app. Ctrl+Tab switches Tulip apps.

PRIVACY / NETWORKING
--------------------
- No account, password, email, OAuth, or login API
- Anonymous locally generated ship identity
- Station chat is manual and stays on the TulipWars PHP relay
- TulipWars never calls world.post_message()
- Tulip World is read-only inside the game

SERVER
------
The installer is already configured for the official TulipWars relay:
https://elementalspark.com/tulipwars2026/api.php

VERSION
-------
TulipWars 2026 v0.1.7
Public Tulip Web test package
July 15, 2026

```
