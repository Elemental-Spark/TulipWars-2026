# TulipWars 2026 v0.1.7

This is the public Tulip Web test release.

## Included

- Complete one-file Tulip Web installer
- Full readable Tulip/MicroPython source
- PHP website, anonymous relay, and dashboard source
- Discord-ready user test package
- Features, controls, installation, and feedback documents
- SHA-256 checksums

## Important status

Physical Tulip CC hardware installation remains on hold until the Tulip Web build is confirmed stable.

## Privacy and Tulip World behavior

- No login API
- No account, password, email, OAuth, or cookie requirement
- Anonymous locally generated ship identity
- Manual station chat is stored on the self-hosted TulipWars relay
- Tulip World is read-only inside the game
- No automatic Tulip World messages or gameplay-event posts
